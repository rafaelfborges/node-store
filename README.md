# Node Store

Simples API desenvolvida no curso [Criando API com Node](https://app.balta.io/courses/1972) do [Andre Baltiere](https://balta.io).

###### Instalação:
```sh
$ cp .env.example .env
$ yarn install or npm install
```

###### Uso:
```sh
$ yarn start or npm start
```
